## Descripción

Este proyecto implementa algoritmos de identificación y control adaptativo para el control de un aeropéndulo. El trabajo combina técnicas avanzadas de control automático con experimentación práctica.

## Estructura del Proyecto

control_adaptativo/   
├── Notebooks/ # Notebooks de Jupyter con análisis y desarrollo    
├── Ensayos/ # Codigo para los ensayos experimentales  
├── Presentaciones/ # Presentaciones del proyecto  
├── mediciones/ # Datos de mediciones y calibraciones  
└── README.md # Este archivo


## Lenguajes

- **Python**: Análisis y procesamiento de datos
- **Jupyter Notebook**: Desarrollo interactivo y documentación
- **C++**: Implementación de algoritmos de control en tiempo real
